The mett text editor
====================

mett is a terminal text editor written in C.
It is made to resemble suckless software.

Requirements
------------
You need GNU Make, ncurses and a C compiler in order to build mett.

Installation
------------
Navigate to the source folder and run
	make install
as root (if necessary).

Configuration
-------------
You can configure most aspects of mett by editing config.h and recompiling
(some knowledge of C is required).
